To get bcid running on your own server, do the following:

1. Get an EZID account and password.  See http://nt2.net/ezid for more information
2. Install mysql somewhere accessible on the internet
3. Copy bcidsettings.template to bcidsettings.props and change appropriate values
3. Copy web/WEB-INF/database-template.xml to web/WEB-INF/database.xml
