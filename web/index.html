BiSciCol

BiSciCol (Biological Science Collections) is an NSF-funded collaborative project with the goal of building an infrastructure designed to tag and track scientific collections and all of their derivatives.

BiSciCol Tools

The following tools from the BiSciCol project are made available to the biodiversity informatics user community to promote linking collections-based data and are provided "as is" without warranty of any kind, either expressed or implied. Questions regarding these tools may be directed to the owners and commmitters of the code-base related to each project below.

Biocode Commons Identifiers, or "BCID" provides tools for creating identifiers. These identifiers are useful both globally and locally, for individuals and organizations, and will satisfy the most demanding requirements related to resolution and persistance. Developers can view the online Javadocs.

The Triplifier converts Spreadsheets, databases, and Darwin Core Archives into RDF/N3 files suitable for use in the Semantic Web and ingestion into the BiSciCol system.

Biocode FIMS incorporates components of the BCID and triplifier technologies to publish sample-based spreadsheet data to an online graph-based database.

BiSciCol uses a set of terms for describing relationships and concepts.
