BiSciCol

BiSciCol (Biological Science Collections) Tracker is a funded NSF collaborative project with the goal of building an infrastructure designed to tag and track scientific collections and all of their derivatives.

BiSciCol Tools

All of the BiSciCol tools are in active development which means that until mid-way through 2013, you will expect to see certain broken links and perhaps not complete functionality. We have posted these tools here and make them available during our development process so we can get feedback from the community and our testers.

BCID provides identifier tools to create awesome identifiers. These identifiers are useful both globally and locally, for individuals and organizations, and will satisfy the most demanding requirements related to resolution and persistance. Developers view the online Javadocs.

The Triplifier converts Spreadsheets, databases, and Darwin Core Archives into RDF/N3 files suitable for use in the Semantic Web and ingestion into the BiSciCol system.

The BiSciCol application lets users visualize graph connections of instances (Identifiers) contained in BiSciCol.

BiSciCol Terms describe how BiSciCol handles relations between Instances.
